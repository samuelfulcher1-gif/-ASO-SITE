# Fulcher ASO Site

## Project Overview
The Fulcher ASO site is designed to provide a seamless user experience with a focus on delivering content related to Fulcher's services and offerings.

## Setup Instructions
1. **Clone the Repository**  
   Run the following command to clone the repository:  
   ```bash  
   git clone https://github.com/samuelfulcher1-gif/-ASO-SITE.git  
   cd -ASO-SITE  
   ```  

2. **Install Dependencies**  
   Make sure you have Node.js installed. Then install the project dependencies:  
   ```bash  
   npm install  
   ```  

3. **Run the Development Server**  
   To start the development server, run:  
   ```bash  
   npm run dev  
   ```  

## Deployment to Vercel
To deploy the project to Vercel, follow these steps:
1. Go to [Vercel](https://vercel.com/) and sign in.
2. Click on 'New Project'.
3. Import your GitHub repository.
4. Follow the on-screen instructions to deploy.
5. After deployment, visit [www.fulcher.com.br](http://www.fulcher.com.br) to see the live site.

## Development Commands
- `npm run dev`: Starts the development server.
- `npm run build`: Builds the application for production.
- `npm run start`: Starts the application in production mode.

Enjoy building your project!
